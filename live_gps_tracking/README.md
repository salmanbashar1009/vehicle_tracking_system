# Live Multi-Vehicle GPS Tracking System

## Features
- Clean Architecture with BLoC
- Firebase Firestore backend
- Simulated live GPS tracking
- Driver and User interfaces
- flutter_map integration

## Setup
1. `flutter pub get`
2. `flutter pub run build_runner build --delete-conflicting-outputs`
3. Configure Firebase project
4. `flutter run`

## Architecture
Follows the specified structure with separate BLoCs for each feature.
